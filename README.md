## 1 适用范围

本手册旨在提供适用于Debian系统的使用说明和配置说明，该手册适合有简单linux基础的人阅读。  

限制：本手册内容适用于Debian13，因本人能力所限，当前手册桌面环境相关内容主要局限于GNOME桌面环境。
